# ♠️ Juego 21 | Modern Edition

Una versión moderna y elegante del clásico juego de cartas "21" (Blackjack simplificado), construida enteramente con tecnologías web tradicionales (Vanilla).

## 🚀 Tecnologías Usadas

- **HTML5**: Estructura semántica moderna y accesible.
- **CSS3**: Diseño profesional implementado con variables (`:root`), Flexbox para una disposición adaptativa (responsive design), transiciones suaves, transformaciones 3D parea el giro de las cartas y modernos efectos de _Glassmorphism_ (cristal esmerilado).
- **JavaScript (Vanilla)**: Lógica del juego, manipulación directa del DOM, gestión de eventos y control coordinado de las animaciones visuales.
- **Recursos Externos**: Empleo de **Google Fonts** (familia moderna _'Outfit'_) para una tipografía limpia y **FontAwesome** para aportar iconografía escalable e interactiva.

## 🎮 Cómo Jugar

El objetivo central del juego es solicitar cartas para sumar puntos y acercarse lo máximo posible al número **21**, ¡sin pasarse!

1. **Añadir Carta**: Al iniciar, haz clic en el botón principal **"Añadir carta"** para robar una carta al azar de la baraja.
2. **Sistema de Puntuaciones**:
   - Las cartas numéricas valen su propio valor (del 2 al 10).
   - Las figuras (J, Q, K) tienen un valor de **10 puntos**.
   - El As (A) tiene un valor de **1 punto**.
3. **Condición de Victoria**: Si tu puntuación acumulada alcanza exactamente los **21 puntos**, ¡habrás ganado la partida instantáneamente!
4. **Condición de Derrota**: Si la suma de las cartas que vas acumulando supera el límite de los 21 puntos, perderás la manga automáticamente.
5. **Volver a jugar**: Cuando finalices una partida (ya sea por victoria o derrota), aparecerá una ventana modal. Solo debes hacer clic en **"Jugar de nuevo"** para que el tablero se reinicie y puedas probar suerte otra vez.

## 💻 Instalación y Uso

Dado que es un proyecto **Vanilla**, no requiere de _frameworks_, tiempos de compilación ni dependencias complejas (como Node.js o npm).

1. Clona este repositorio en tu equipo local:
   ```bash
   git clone https://github.com/carlosmorenodev/juego21.git
   ```
2. Entra en el directorio recién clonado:
   ```bash
   cd juego21
   ```
3. Simplemente abre el archivo `index.html` en tu navegador web de preferencia (Chrome, Firefox, Safari, Edge) o utiliza una extensión tipo _Live Server_ en tu editor de código.
